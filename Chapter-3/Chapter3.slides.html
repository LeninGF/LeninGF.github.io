Clasificando especies de Iris¶

Se desea distinguir entre 4 tipos de flores iris. Para esto se registran:

  • largo del ptalo y ancho del ptalo
  • largo del spalo y ancho del spalo

Las clases a distinguir son:

  • setosa
  • versicolor
  • virginica

Cargando el dataset de Iris

In [2]:

El dataset viene como un diccionario. Las claves del diccionario de iris son:

Se va a resolver este problema con el enfoque supervisado de Machine Learning, ya que disponemos las medidas (longitud, ancho) del ptalo y el spalo de cada tipo de flor.

Las medidas registradas son el vector de caractersticas $\mathbf{X}$

Las clases identificadas por el experto (setosa, versicolor, virginica) son $\mathcal{Y}$

Resumiendo:

Se desea encontrar una funcin $g: \mathcal{X} \rightarrow \mathcal{Y}$ que aproxime a la funcin verdadera $f: \mathcal{X} \rightarrow \mathcal{Y}$. Donde:

$\mathbf{X} = \{longitud\_sepalo, ancho\_sepalo, longitud\_petalo, ancho\_petalo\}$

$\mathcal{Y} = \{setosa, versicolor, virginica\}$

Por tanto se tiene un conjunto de datos $\mathcal{D}$ tal que se forma de parejas $(\mathbf{x}_{1},y_{1}), (\mathbf{x}_{2},y_{2}) \dots (\mathbf{x}_{N},y_{N}) $

Evaluación de Desempeño del algoritmo de aprendizaje¶

Para evaluar el desempeo del algoritmo entrenado, es necesario separar el dataset $\mathcal{D}$ en dos juegos de datos:

  • Training Set
  • Validation Set
  • Test Set

En la mayora de casos se suele utilizar nicamente los 2 primeros. El Validation set se utiliza para realizar mejoras a los hiperparmetros del modelo de tal manera que se mejore la funcion de costo y la precision(accuracy) del modelo. El Test Set en cambio es un conjunto de datos a los que el modelo nunca esta expuesto y que sirve para medir la capacidad de generalizacin del mismo frente a nuevos datos

Mtodo train test split: https://scikit-learn.org/stable/modules/generated/sklearn.model_selection.train_test_split.html

K-Nearest Neighbors¶

  • Asume que toda instancia corresponde a puntos en un spacio $n$ - dimensional $R^{n}$
  • Los vecinos ms cercanos se definen en trminos de una medida de distancia e.g. La distancia Euclidea.

Por ejemplo, si una muestra de $\mathcal{X}$ del dataset $\mathcal{D}$ es un vector de caractersticas:

$\mathbf{x}_{r} = <x_{r,1},x_{r,2},\dots, x_{r,n}>$, entonces, la distancia entre dos datos $i$ y $j$ es:

$d(\mathbf{x}_{i}, \mathbf{x}_{j})=\sqrt{\sum_{k=1}^{n}({x}_{i,k}-{x}_{j,k})^{2}}$

  • La funcin objetivo (target function) en KNN puede ser discreta(clasificacin) o real(regresin)

Algoritmo KNN¶

Training:

  • Para cada ejemplo de entrenamiento $(\mathbf{x}, y)$, aada el ejemplo a la lista de entrenamiento.

Clasification: Dada una instancia $x_{q}$ a ser clasificada

  • Defina un valor de $k$ vecinos prximos. Teniendo $\mathbf{x}_{1}, \mathbf{x}_{2} \dots, \mathbf{x}_{n} $ muestras del conjunto de ejemplos de entrenamiento y calcule su distancia a $q$
  • Devuelva el voto de la mayora

$g(\mathbf{x}_{q}) = argmax \sum_{i=1}^{k}\delta(y_{verdadero}, f(\mathbf{x}_{i}))$

Donde: $\delta(y_{verdadero}, f(\mathbf{x}_{i})) = 1$ si $y_{verdadero}=f(\mathbf{x}_{i})$ y $\delta(y_{verdadero}, f(\mathbf{x}_{i})) = 0$, caso contrario
